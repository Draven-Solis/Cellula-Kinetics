# Project Genetrix 🧬

## About the Project
- An interactive visualization guiding users through meiosis, gamete formation, and fertilization.
- Designed to help students and curious learners visualize genetic processes step-by-step.

## About the Creator
- I am a biology lecturer with no formal coding background.
- I used AI tools—Replit, Claude AI, DeepSeek, OpenAI, and Gemini—to design, test, and refine this project.
- Each step, from conceptualization to troubleshooting, was powered by AI, allowing me to bring this vision to life.

## Key Features
- Step-by-step animations of meiosis I & II.
- Gamete formation and chromosomal behavior.
- Fertilization sequence and early zygote development.
